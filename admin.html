Мои работы

Управляй работами прямо в Google таблице — изменения сразу появятся на сайте

💡 Сайт читает данные напрямую из твоей Google таблицы. Чтобы добавить работу — просто добавь строку на лист Works. Чтобы удалить — удали строку. Сайт обновится автоматически при следующем открытии.

Быстрое добавление работы

1

Загрузи файл на Google Drive

Видео или фото → загружаешь в любую папку на drive.google.com

2

Открой доступ к файлу

Правая кнопка → «Открыть доступ» → «Все у кого есть ссылка» → Читатель → Готово

3

Скопируй ссылку

«Копировать ссылку» — выглядит как: https://drive.google.com/file/d/XXXXX/view

4

Добавь строку в таблицу

Открой Google таблицу → лист Works → новая строка с названием, категорией и ссылкой

Настройка таблицы

Как правильно заполнить Google таблицу чтобы сайт всё подхватил

В таблице должно быть 4 листа: Works, Settings, Experience, Tools. Первая строка каждого листа — заголовки колонок (именно такие как указано ниже).

Лист: Works — работы

КолонкаПримерОписание
title обязательноРекламный ролик NikeНазвание работы
src обязательноdrive.google.com/file/d/XXXСсылка на файл (Drive, YouTube, Vimeo)
category обязательноРекламаКатегория (используется для фильтров)
type необязательноvideovideo или photo (по умолчанию video)
year необязательно2024Год проекта
client необязательноNike RussiaКлиент или бренд
desc необязательноКороткий ролик для InstagramОписание проекта

Скопируй заголовки для первой строки листа Works:

title src category type year client desc

Настройки сайта

Все настройки меняются в листе Settings таблицы

Чтобы изменить имя, контакты или тексты — открой таблицу → лист Settings → найди нужный ключ → измени значение. Сайт подтянет изменения при следующем открытии.
